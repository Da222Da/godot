::: tip 什么是二维向量？

二维向量，用于表示二维空间中的方向和大小。记作（x，y），如下图所示：

![二维向量](/images/vector.png)

其中，“m”表示大小，即向量的长度。“d”是水平轴的角度，这是表示方向的视觉方式。

:::

::: warning 二维向量的运算常用方式:

1. **加法：** (1, 2) + (3, 4) = (4, 6)
2. **减法：** (1, 2) − (3, 4) = (−2,−2)
3. **与标量相乘：** 就是与固定数字相乘，3 × (1, 2) = (3, 6)

![二维向量运算](/images/vector2-operation.png)

:::
