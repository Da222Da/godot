::: tip 游戏、场景与节点之间的关系？

一个场景 == 根节点 + N 个子节点

一个游戏 == N 个场景

:::

在 Godot 中，节点是构建游戏最基本的元素，例如，播放一段音频、显示一张图像等等。而 Godot 引擎给我们提供了上百种不同节点，帮助我们构建游戏的各个部分。

而场景则是将拥有不同功能的节点组织起来，从而实现游戏需求。所以，场景又被称为节点树（如下图所示）。

![节点树](/images/node_tree.png)

::: danger 注意事项

值得注意的是，一个场景只有一个父节点(称为根节点)，但可以有任意数量的子节点。

:::
