# GDScript 脚本是什么？

`GDScript` 是 Godot 引擎官方自定义的一种**脚本语言**，它能赋予游戏逻辑，让我们游戏动起来。

::: warning 注意！

每一个 GDScript 文件都代表着一个“子类”。

这意味着你在文件范围内声明的所有内容(例如变量和函数)都是类定义的一部分，并且对其他文件不可全局使用。

:::
