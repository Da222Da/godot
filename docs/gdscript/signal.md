::: tip 什么是信号？

在 Godot 中，`信号(Signal)`是一种用于在不同节点(或场景)之间进行通信的机制(如下图所示)。这种机制使得对象之间的交互更加灵活和解耦。

![信号机制](/images/signal.drawio.svg)

:::

::: warning 内置信号与自定义信号的区别？

-   `内置信号` 是 Godot 直接提供给我们的，开箱即用。往往用于同一场景下，不同节点之间的通信。
-   `自定义信号` 是用户自定义的，用户根据自己的实际需求来扩展节点的功能。它能让场景发出信号，往往用于不同场景之间的节点通信。

:::
