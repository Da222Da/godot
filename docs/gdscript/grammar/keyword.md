关键字就是编程语言中具有特定的含义的标识符，具体如下：

| 关键词 |     说明     |               示例                |
| :----: | :----------: | :-------------------------------: |
|  var   |   声明变量   |     如`var number: int = 100`     |
| const  |   声明常量   |    如`const number: int = 100`    |
|  func  |   声明函数   |   如`func _process() -> void:`    |
|  pass  | 忽略函数主体 | 如`func _process() -> void: pass` |
