# 用户输入

在 Godot 中，用户输入是指玩家通过键盘、鼠标、手柄或触摸屏等设备输入的操作`如按键、点击或滑动等`，与游戏进行互动。而 Godot 引擎要做的就是检测和响应用户输入。

::: tip 用户输入 —— 知识结构

#### 基础概念

1. **输入映射**：将输入设备（键盘、鼠标等）的输入事件（如“按下空格键”）映射到游戏操作中。

#### 内置对象

1. Input 单例类: 用于检测和响应各种输入事件
    - `Input.is_action_pressed(action)` 监听键盘的某键是否被按下
