# 用户输入

用户输入是指玩家通过输入设备`键盘、鼠标、手柄或触摸屏等`与游戏进行互动的过程。

而 Godot 引擎提供了一系列强大的功能，帮助我们检测、并响应用户输入。

::: tip 用户输入 —— 知识结构

1. Input 单例类: 用于检测和响应各种输入事件
    - `Input.is_action_pressed(action)` 监听键盘的某键是否被按下

:::
