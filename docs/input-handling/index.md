# 用户输入

在 Godot 中，用户输入是指玩家通过键盘、鼠标、手柄或触摸屏等设备输入的操作，如按键、点击或滑动等。

Godot 引擎提供了方便的输入处理系统，可以轻松地检测和响应用户输入。您可以通过信号或脚本来处理不同类型的用户输入。您还可以使用 Godot 的输入事件功能来管理用户输入。

::: tip 基础知识

#### 基础概念

- **输入映射**：将输入设备（键盘、鼠标等）的输入事件（如“按下空格键”）映射到游戏操作中。
