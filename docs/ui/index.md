# 用户界面

在 Godot 中，用户界面（UI）用于创建和设计游戏的用户界面元素，如按钮、文本标签、滑块等，以便玩家与游戏进行交互。UI 设计可以通过布局编辑器或脚本来实现，使开发者能够定制游戏的外观和交互方式。通过用户界面，玩家可以与游戏世界进行沟通，执行操作并获得反馈。

::: tip

#### 内置节点 Control Nodes

-   `CanvasLayer 节点` 画布层 `默认图层索引为 -1`
-   `Button 节点` 按钮
-   `Label 节点` 用于显示文本
-   `TextureRect 节点` 用于显示图像
-   `VideoStreamPlayer 节点` 用于播放视频

:::
