::: tip TileMap 是什么？

在 Godot 中，TileMap 是一个用于创建和管理 2D 网格地图的。它不仅可以帮助我们快速地构建游戏地图，还可以减少你需要的艺术数量。

:::

::: tip TileSet 是什么？

图块集（TileSet）来定义可用的瓷砖。图块集是一组预定义的图块，可以包含不同的地形、墙壁、障碍物等。您可以从图块集中选择和放置瓷砖来构建地图。

:::

::: warning 快捷键

- TileMap 绘制地图
  - 选中 TileMap 图块之后，然后按住`Shift + 鼠标左键`，可以拖拽出一条直线。
  - 按住`Ctrl 键`，然后点击已经绘制出来的图块，就可以吸取该图块。
  - 按住`Ctrl 键`，然后拖到鼠标左键，就可以吸取多个图块。
  - 按住`Ctrl + Shift 键`，可以拖出一个矩形。
  - 按住鼠标右键，再拖拽就可以删除已绘制出来的图块。

:::
