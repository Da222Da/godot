# 熟悉可视化编辑器

通过前面几期的内容，想必你已经学会了下载并安装 Godot、看到了项目管理器界面、创建第一个项目，并启动可视化编辑器。

为了进一步地熟悉 Godot 的可视化编辑器。接下来，我们将会一起创建一个简单的场景。这个场景将包含一个图像和一个简单的按钮。我们还将添加一个简单的脚本，通过点击按钮来控制图像显示和消失。

<el-card class="box-card" header="创建场景示例的步骤如下：">

1. 构建游戏场景:

   - 在场景区域，点击新增节点按钮之后，你会看到一个新增节点对话框，对话框中列出了所有可用的`节点`。然后。选择其中的`Node2D`节点作为你项目的根节点，并将其重命名为“myFirstScene”。
   - 再次点击新增节点按钮，新增一个 `Sprite` 子节点。然后，将文件系统区域的 `icon.png 图片` 拖到检查器中的 `Texture 纹理属性`处。
   - 选中 `myFirstScene` 根节点，再新增一个 `Button` 子节点。然后在检查器中，将`Text 文本属性`设置为“Click Me”。
   - 最后，让我们将 Sprite 节点和 Button 节点的内容拖到可视区域的中间位置，然后，点击`运行场景`按钮，之后你就能预览到你构建的场景了。

2. 编写脚本：

   - 选中 `myFirstScene` 根节点，在场景区域中，创建`新建脚本`按钮，创建一个名为“myFirstScene.gd”的脚本。
   - 编写脚本内容：

   ```gdscript
   extends Node2D

   func _on_Button_pressed():
      $Sprite.visible = not $Sprite.visible
   ```

3. 点击“运行游戏”按钮，查看效果。

4. 导出项目，发给你的朋友吧。

</el-card>
