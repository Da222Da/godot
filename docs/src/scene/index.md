# 节点与场景

所有的 Godot 游戏是由节点和场景构成的。其中，每个 Godot 游戏可能由无数个**场景**组成，而每个场景又由许多的**节点**以**树状结构**的形式组合而成。

所以说，节点就是 Godot 游戏的基本组成元素。就像人类食谱中的食材一样。Godot 引擎包含很多种节点，可以用来显示图像、播放声音、表示摄像机等等。如图所示：

![节点树](/images/node_tree.png)

其中，每一个节点又拥有属于自己的名称、属性、方法、以及子节点。如果你了解`OOP 面对对象编程`和`数据结构--树状结构`的话，就能轻松地理解 Godot 引擎中的节点和场景了。
