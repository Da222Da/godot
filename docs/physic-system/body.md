# 物体、物理体之间的关系？

::: tip 物体、物理体之间的关系？

物理体，就是受到 Godot 物理系统影响`例如，重力等`的物体。

:::

在 Godot 中，物理学主要基于 **物体(Body)** 和碰撞形状(Shape)。其中，物体（Object）是指在游戏世界中存在的实体。它可以是角色、道具、环境元素等。物体可以具有位置、旋转、缩放等属性，并且可以在游戏中进行移动、交互和与其他物体发生碰撞。

然而，模拟物理世界中的物体被称为**物理体(Physic Body)**，用来模拟物体之间的运动和相互作用以及与环境的相互作用。如果你想定义物理体，可以试试以下节点：

- Static Body（静止体）：在游戏中不移动但与其他对象交互的对象。
- Rigid Body（刚体）：与物理引擎交互并受其影响的对象。
- Kinematic Body (运动体)：不受物理直接影响但需要移动和检测碰撞的对象。
