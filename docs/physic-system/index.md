# 物理系统是什么？

> 在 Godot 中，物理系统是指 Godot引擎中内置的用于处理物理仿真和碰撞检测的模块，可以帮助你创建更真实和交互性的游戏体验。它提供了一套强大的工具和函数，用于模拟物理世界中的物体之间的相互作用。


在 Godot 中，物理学主要基于**物体(Body)**和**碰撞形状(Shape)**。

在 Godot 中，物体（Object）是指在游戏世界中存在的实体。它可以是角色、道具、环境元素等。物体可以具有位置、旋转、缩放等属性，并且可以在游戏中进行移动、交互和与其他物体发生碰撞。

在 Godot 中，形状（Shape）是物体的几何形状或碰撞体积。在物理仿真中，形状用于检测物体之间的碰撞。形状可以是简单的几何体，如盒子、球体、胶囊体，也可以是复杂的自定义形状。 

总之，物体是在游戏世界中存在的实体，而形状是物体的几何形状或碰撞体积，用于进行物理仿真和碰撞检测。您可以为物体添加形状，以便进行物理仿真和碰撞检测。物体的形状可以与其他物体的形状进行交互，从而触发碰撞事件。形状还可以用于确定物体的边界和碰撞区域，以便进行精确的碰撞检测和响应。
