# 场景系统

在游戏开发过程中，我们可以创建一个或多个场景，每个场景都包含了自己的场景内容、脚本逻辑以及游戏资源`例如，声音、动画`。

而我们可以借助 Godot 提供的场景系统来管理这些游戏场景。

::: tip 基础知识

-   **常见操作**
    -   操作场景：
        -   `Ctrl + N`新建一个场景
        -   `Ctrl + Shift + O` 快速查找已有场景
    -   操作节点：
        -   `Ctrl + A` 快速调出创建节点的对话框。
        -   `Ctrl + D` 快速复制当前选中的节点。
        -   `Ctrl + 鼠标左键` 同时选择多个节点。
    -   操作视图：
        -   `F 键` 居中视图
        -   `滚动鼠标滚轮` 缩放视图
        -   `长按鼠标滚轮` 挪到视图

:::
