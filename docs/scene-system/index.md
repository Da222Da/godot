# 场景系统

在游戏开发过程中，我们可以创建一个或多个场景，每个场景都包含了自己的场景内容、脚本逻辑以及游戏资源`例如，声音、动画`。

而我们可以借助 Godot 提供的场景系统来管理这些游戏场景。

::: tip 基础知识

#### 基础概念

-   **节点：** 是构成场景的基本单位，可以是角色、相机、灯光等游戏对象。
-   **场景：** 由无数个节点组成的一种树状结构，所以也称场景树。场景可以被保存到磁盘上，然后被实例化到其他场景中。
