# 场景系统

在游戏开发过程中，我们可以创建一个或多个场景，每个场景都包含了自己的场景内容、脚本逻辑以及游戏资源`例如，声音、动画`。

而我们可以借助 Godot 提供的场景系统来管理这些游戏场景。

::: tip 基础知识

- [场景与节点之间的关系？](./what-scene.md)
- **场景操作：**
  - 预加载场景文件`@GDScript.preload()`
  - 实例化场景对象`PackedScene.instantiate()`
  - 退出游戏场景`Node.get_tree().quit()`
  - 切换游戏场景`Node.get_tree().change_scene_to_file()`
- **节点操作：**
  - 获取节点`$"."`
  - 获取子节点`Node.get_children()`

:::

::: warning 常用快捷键

#### 场景操作的快捷键

| 名称       | 说明         |
| ---------- | ------------ |
| `Ctrl + N` | 新建一个场景 |

#### 节点操作的快捷键

| 名称              | 说明                       |
| ----------------- | -------------------------- |
| `Ctrl + A`        | 快速调出创建节点的对话框。 |
| `Ctrl + D`        | 快速复制当前选中的节点。   |
| `Ctrl + 鼠标左键` | 同时选择多个节点。         |
