# 如何创建一个场景？

1. 如果你想创建一个场景的话！首先就需要首先创建一个新的项目。打开项目后，你看到的应该是一个空的编辑器。

![Godot编辑界面](/images/Godot编辑界面.png)

2. 在空场景中，点击左侧“场景”面板左上角的“添加”按钮, 添加一个 Node2D 节点作为根节点，并将其重命名为“MyFirstScene”。

![新增根节点](/images/create_root_node.png)

3. 然后，在“场景”面板中，点击“添加”按钮，添加一个“Label”子节点。

![新增 Label 节点](/images/create_label_node.png)

4. 设置 Label 文字内容为“Hello World”，并将其位置设置为居中。

![设置属性](/images/set_label_attr.png)

5. 保存场景，然后点击“运行”按钮，查看场景效果。

![run](/images/run_sence_button.png)
