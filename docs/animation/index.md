# 动画

动画是大多数游戏的一个关键方面。它被用来模拟跑步、跳跃、抓取、射击以及随着时间的推移发生的更多动作。即使角色静止不动，通常也会给它一个空闲（ idle）的动画，在动画中，角色上下移动，给人一种玩家正在呼吸的感觉。

::: tip 基础知识

#### 基础概念:

- **精灵帧动画**：快速播放每一帧画面形成的动画。
- **关键帧动画**：先设置关键性的画面帧，至于关键帧之间的画面帧则靠算法生成，然后快速播放这些画面帧形成的动画。
