::: tip 什么是关键帧动画？

关键帧动画是现在最常用的一种动画技术。它允许您在时间轴上指定对象的属性值。当您播放动画时，对象的属性将在关键帧之间平滑过渡。

而**关键帧则是动画中特定时间点的状态** 。当您添加关键帧时，你可以指定对象的属性在该时间点的值。当您播放动画时，对象的属性将在关键帧之间平滑过渡。

![keyframe-animation.png](/images/keyframe-animation.png)

:::

::: danger (个人经验)我们该什么时候给对象添加属性关键帧呢？

1. **不同时间点展示不同画面**时，给每一个时间点添加一个关键帧。
2. **对画面有影响的属性值**发生变化的情况下，给该属性添加一个关键帧。

:::

::: warning 不同属性之间的关键帧有什么联系？

在动画中，不同属性之间的关键帧可以用来控制对象在不同属性之间的平滑过渡。通过在不同时间点添加关键帧，您可以指定对象在这些时间点上的属性值，然后动画系统会自动在关键帧之间进行插值，以实现平滑的过渡效果。

例如，假设您有一个 Sprite2D 节点，您想要在动画中同时控制其位置和旋转。您可以在 AnimationPlayer 中为每个属性添加关键帧，然后在播放动画时，位置和旋转将同时变化，并且系统会自动计算它们之间的过渡。

**关键帧之间的联系是通过动画系统的插值算法实现的。** 该算法根据关键帧之间的`时间差`和`属性值`的差异，计算出每一帧的属性值。这样，您就可以创建出平滑的过渡效果，使对象在动画中流畅地变化。

总结起来，不同属性之间的关键帧通过动画系统的插值算法联系在一起，以实现平滑的属性过渡效果。

:::
