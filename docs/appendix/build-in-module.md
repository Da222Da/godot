# 内置模块

- [Node 基类](./modules/node.md)，所有 Godot 场景对象的基类。
    - [Area2D 碰撞区域类](./modules/area2d.md)，用于检测到其他 2D 碰撞对象是否进入或退出该区域。
- [Input 单例类](./modules/input-singleton.md) 用于检测和响应各种输入事件，包括键盘、鼠标、触摸屏、游戏手柄等.
- [Vector2 类](./modules/node.md)，用于创建和计算二维向量。
- Resource，用于加载资源，包括图片、字体、音频、视频等。
    - [PackedScene](./modules/packedScene.md) 用于对场景资源本身进行的操作和检查。
