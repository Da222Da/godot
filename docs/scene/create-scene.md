# 如何创建一个游戏场景？

<ImageTextCard>
<template #content>

1. 首先创建一个新项目。打开项目后，你会看到一个空的编辑器。
2. 点击“添加子节点”按钮，你会看到一个“创建节点”弹框。
3. 在搜索框中输入我们将要创建的节点名称`Label`，双击 Label 节点将其选中，这样我们项目的根节点就被创建好了。
4. 接下来，前往视口右侧的“检查器”面板区域。点击 Text 属性的文本输入框，然后填入“Hello World”。
5. 前往编辑器的 2D 可视区域，用鼠标左键按住 Label 节点的可视化内容，将其拖动到视口中间位置。好了，场景内容就创建完毕了。
6. 最后，让我们一起来测试一下吧。点击屏幕右上角的“运行场景”按钮，你会看到一个“保存场景”的弹框。点击“保存”之后，程序会打开一个新窗口，显示“Hello World”字样。Nice!
