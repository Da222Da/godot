# 游戏、场景与节点之间的关系？

::: tip 游戏、场景与节点之间的关系？

一个游戏 == N 个场景

一个场景 == 根节点 + N 个子节点

N 个节点组成了一个游戏
:::

在 Godot 中，节点是构建游戏最基本的元素。就像人类食谱中的食材一样。Godot 引擎包含很多种节点，可以用来显示图像、播放声音、表示摄像机等等。

但是，但个节点本身不能做很多事情。为了充分挖掘其潜力，需要将其与其他节点结合起来，这样就形成了场景。

一个场景只不过是一个节点，加上它的子节点们，形成一个树形结构。值得注意的是，一个场景只有一个父节点(称为根节点)，但可以有任意数量的子节点（如下图所示）。
